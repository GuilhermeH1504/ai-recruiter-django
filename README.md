# 📄 Análise de Currículos com IA

Aplicação web desenvolvida com **Python + Django** que utiliza inteligência artificial para analisar currículos e avaliar candidatos para vagas específicas.

## 💡 Sobre o Projeto

A ferramenta permite que profissionais de RH façam o upload do currículo de um candidato em PDF, informem a vaga desejada e recebam uma análise automática com os **pontos fortes e fracos** do candidato para aquela posição.

## 🚀 Tecnologias Utilizadas

- **Python** + **Django** — backend e estrutura web
- **Groq API** — processamento de linguagem natural com IA
- **PyPDF** (ou similar) — leitura e extração de texto de PDFs

## ⚙️ Funcionalidades

- Upload de currículo em formato PDF
- Informar a vaga desejada
- Análise automática via IA
- Retorno dos pontos fortes e fracos do candidato

## 📚 Aprendizados
